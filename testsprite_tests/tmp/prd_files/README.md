# Job Portal Backend

This is the Node.js + Express + PostgreSQL backend for the Job Portal project.

## **Setup Instructions**

1. **Clone the repository**
```bash
git clone https://github.com/SEMESTER-5-TEAM/CloudComputing.git
cd CloudComputing/backend
```

2. **To checkout the database**

Code the given one's in the schema.sql 
also do create an .env file within the backend folder where,

```bash 
PORT=5000
DB_USER=postgres
DB_PASSWORD=your_password_here
DB_HOST=localhost
DB_NAME=jobportal
DB_PORT=5432
JWT_SECRET=your_secret_key_here
```
